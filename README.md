# 🎵 Melodia

A minimalist, offline-capable music player with mood-based organization. No server required, no data collection, just pure music enjoyment.

## ✨ Features

- 🎨 **Minimalist Design** - Clean, distraction-free interface
- 📱 **Progressive Web App** - Install on any device like a native app
- 🔌 **Offline First** - Works without internet after first load
- 🎭 **Mood Organization** - Categorize songs by Chill, Hype, Focus, or Sad
- ❤️ **Favorites** - Quick access to your most-loved tracks
- 🔍 **Smart Search** - Filter by song name or artist instantly
- 🎵 **Drag & Drop** - Easy file management
- 🔒 **Privacy Focused** - Everything stays local in your browser

## 🎯 Supported Formats

MP3 · FLAC · AAC · OGG · WAV · M4A · OPUS

## 📱 Installation

### On Desktop
Just visit the site and start using it! Or click the install icon in your browser's address bar.

### On Mobile

**iOS (iPhone/iPad)**
1. Open in Safari
2. Tap Share button
3. Select "Add to Home Screen"
4. Enjoy!

**Android**
1. Open in Chrome
2. Tap menu (⋮)
3. Select "Install app"
4. Done!

## 🚀 Quick Start

1. Click **"+ ADD"** button or drag audio files anywhere
2. Organize songs with mood tags (optional)
3. Press play and enjoy!

## 🛠️ Tech Stack

- Pure HTML5/CSS3/JavaScript
- No frameworks or dependencies
- Service Workers for offline functionality
- Web Audio API for playback
- Local storage for all data

## 🎨 Features Detail

### Player Controls
- Large play/pause button with intuitive icons
- Smooth volume slider
- Animated visualizer bars
- Previous/Next track navigation
- Progress seeking

### Organization
- **All** - View entire library
- **Favorites** - Quick access to starred songs
- **Mood Tabs** - Filter by Chill, Hype, Focus, or Sad
- **Search** - Real-time filtering

### Smart Features
- Auto-detects song metadata from filenames
- Queue management with current position
- Visual feedback for currently playing track
- Smooth animations and transitions
